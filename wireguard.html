Instructions to get yourself connected to the server we are going to use for hosting our Minecraft server and servers for other occasional games

Windows

  1. Download and install Wireguard from https://www.wireguard.com/install/ and then run it
  2. On the main window, click the small arrow beside the 'Add Tunnel' button, and then click 'Add Empty Tunnel' (or press Ctrl+N)
  3. Contact and give vibbe or ninkeo your public key (located under the "Name" box) and ask for what local ip to use.
  4. Paste the following text into the window that appears below the [Interface] and PrivateKey lines (DO NOT OVERWRITE THE LINES GENERATED BY WIREGUARD):
    5. Address = 192.168.55.[1]/32

[Peer]
PublicKey = a9T+Tp02KzhDxsgcn3gFGh1d6JfnE1WTRyMf4kgq6Ec=
AllowedIPs = 192.168.55.0/24
Endpoint = [2]

Linux

  1. Install wireguard through your package manager. https://www.wireguard.com/install/ . After that, restart your computer so the kernel module loads properly.
  2. From now on everything should be run as root so so it is a good idea to do $ sudo su then # cd /etc/wireguard
  3. Generate a private and public key: 
    # umask 077
# wg genkey > dumpster
# wg pubkey < dumpster > dumpster.pub
    Read both your keys with $ cat or anything else, you will need them.
  4. Contact and give vibbe or ninkeo your public key and ask for what local ip to use.
  5. Create a file called dumpster.conf here, in /etc/wireguard, with the following contents: 
    [Interface]

Address = 192.168.55.[1]/32
Privatekey = **your private key**

[Peer]
PublicKey = a9T+Tp02KzhDxsgcn3gFGh1d6JfnE1WTRyMf4kgq6Ec=
AllowedIPs = 192.168.55.0/24
Endpoint = [2]
  6. With the config saved you can turn up the network at any time with sudo $ wg-quick up dumpster (running it as root is important) and turn it down when you're done with $ sudo wg-quick down dumpster.

[1] Represents the network number that has been assigned to you by an admin.


[2] The hostname that you will be able to find in the pinned messages of the Discord server (#gaming-general).
